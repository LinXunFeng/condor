## 命令

```shell
flutter pub run build_runner build
```

全局激活安装

```shell
dart pub global activate --source path 'xx/xxx/condor/packages/condor_cli'
```

编译成可执行文件

```
dart compile exe bin/condor.dart
```

相关说明: [dart-compile](https://dart.dev/tools/dart-compile)
